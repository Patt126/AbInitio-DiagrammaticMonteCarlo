# Breathing Coupling Mode

Main executable source: `Breathing_Coupling/breathing.cpp`

Default outputs:

- `Breathing_Coupling/results/breathing.csv`
- `Breathing_Coupling/results/breathing_green.csv`

Plot script:

- `Breathing_Coupling/plot_breathing.py`
  - default input: the two files above
  - output:
    - `Breathing_Coupling/results/breathing.png`
    - `Breathing_Coupling/results/breathing_green.png`

Run from project root:

```bash
make run-breathing
```
